
# Quantum-Based Intrusion Detection System (Q-IDS)

This project implements a hybrid quantum-classical intrusion detection system using quantum machine learning.

## Features
- Data preprocessing
- PCA-based feature reduction
- Quantum encoding
- Variational Quantum Circuit (VQC)
- Performance evaluation

## How to Run
1. Install dependencies: pip install pennylane pandas scikit-learn numpy
2. Place your dataset as dataset.csv
3. Run: python main.py
